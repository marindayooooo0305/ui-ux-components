# ui-ux-components

个人 UI 组件与 UX 动效收藏库。

## 目录结构

```
components/
├── buttons/       # 按钮组件
├── toggles/       # 开关 / 切换组件
│   └── pill-toggle.html
├── inputs/        # 输入框组件
├── cards/         # 卡片组件
├── modals/        # 弹窗 / 对话框组件
└── navigation/    # 导航组件
effects/           # CSS 动效 / 交互动画
utils/             # 通用工具 / 帮助函数
```
